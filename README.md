# TeamFlow API

TeamFlow API is a backend SaaS application built with FastAPI for managing teams, workspaces, projects, and tasks.

##  Features
- FastAPI backend
- Environment-based configuration (.env)
- Modular project structure
- Ready for PostgreSQL integration
- Clean architecture (api, core, db, models, services)

##  Tech Stack
- Python
- FastAPI
- Pydantic
- SQLAlchemy (coming next)
- PostgreSQL (planned)
